# Rest 서비스
HTTP URI + HTTP Method
- HTTP URI를 통해 자원을 명시
- HTTP Method를 통해 자원을 제어하는 명령


|Http Method|CRUD|
|---|---|
|POST|Create(Insert)|
|GET|Read(Select)|
|PUT|Update or Create|
|DELETE|Delete|


## Restful
Rest의 원리를 따르는 시스템

## Rest API 설계 규칙
* URI는 정보의 자원을 표시
* 자원은 명사 사용
* 자원의 행위는 HTTP Method(GET, PUT, POST, DELETE 등)로 표현
