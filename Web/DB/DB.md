# Database

## DBMS (데이터베이스 관리 시스템)
* 사용자와 데이터베이스 사이에서 질의처리와 접근관리
* 장점
    * 데이터의 중복성과 불일치성 감소
    * 보안
    * 백업과 복구
    * 일관된 데이터 유지

## Schema (스키마)
* 데이터베이스의 `구조`와 `제약조건`에 대해서 기술한 것

## 데이터베이스 언어
* DDL (데이터 정의어)
    * 데이터 간의 관계를 정의하거나 변경, 수정할 때 사용
    * 데이터베이스 관리자나 설계자가 사용
* DML (데이터 조작어)
    * 데이터를 검색, 수정, 삭제 등을 수행할 때 사용
    * 데이터베이스 사용자가 사용
* DCL (데이터 제어어)
    * 데이터의 보안, 무결성, 복구, 병행 수행 제어 등을 정의할 때나 사용자 권한 설정할 때 사용
    * 데이터베이스 관리자가 사용
